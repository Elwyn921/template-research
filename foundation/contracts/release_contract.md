# Release Contract

A downstream release should identify:

- Foundry release and exported capability manifest;
- active project `AGENTS.md` and `TOOLFLOW.md`;
- source registry and evidence matrix;
- dataset inventory;
- experiment or run records;
- figure specs and generated assets;
- manuscript or memo source;
- citation checks;
- unresolved caveats.

Foundry validators may audit readiness. They do not approve downstream release
decisions.
